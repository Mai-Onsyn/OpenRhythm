# OpenRhythm 🎹

> 🚧 **Status: Active Development** — Expect bugs, missing features, and rapid improvements!

## What is this?

OpenRhythm is an open-source MIDI visualization and practice software. Our ultimate goal is to be the **ultimate open-source alternative to Synthesia** — free, transparent, and fully under your control.

## Cross-Platform

- 💻 JVM (Such like Windows, Ubuntu, macOS)
- 📱 Android
- 📱 iOS (Planned)

Powered by **Kotlin Multiplatform** + **Compose Multiplatform**.

## Current Features

- 🎵 MIDI Waterfall Flow Playback
- 📂 MIDI File Management & Preview

## Future Plans

> *In short: Everything Synthesia has, we want it too — and we want it **open**!*

- Free Practice Mode
- Follow-along Practice Mode
- Basic music practice utilities (loop, speed adjustment, etc.)
- Fully customizable color schemes (want a "dazzling black" with rainbow edges? You got it!)
- Adjustable waterfall styles and visual effects

## Quick Start (Build & Run)

Just fire it up from the project root:

```bash
./gradlew build run
```
Compose Multiplatform will handle all the dependencies for you. No extra setup required.
